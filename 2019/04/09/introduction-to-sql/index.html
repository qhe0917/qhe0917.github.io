2019-04-09

I was planning to write an introduction about SQL to include some basic information about it, what does it do, what is the differences between other programming languages, and what is the advantages. But after I opened the wiki page, there are so many new terms in the description, each needs a further explanation for me to understand the content. Then I realize, it’s almost impossible to understand all these terms unless I learn this language first and process some datasets with different languages. It is like I don’t need all those concepts to learn R, and only after I have used R for a while until I can really understand the pros and cons of using R for data processing. So, I plan to study to SQL first and answer those questions later in this blog.
